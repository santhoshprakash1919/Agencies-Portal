# Sindhu Agencies Bulk Order Portal

## Overview
Sindhu Agencies is a wholesale distributor located in Kaveripattinam, Krishnagiri District. This project is a simple 3-page website built using **HTML, CSS, and JavaScript** to showcase products and allow customers to place bulk orders. The system generates invoices in PDF format for every order.

## Usage
1. Navigate to the **Products** page to view available items.  
2. Place a bulk order via the **Order page**.  
3. Download the generated invoice in PDF format.
